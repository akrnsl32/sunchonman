This is a collection of birds.

, Harpy eagle(JSBirds/HarpyEagle), Vogelkop bower bird(JSBirds/BowerBird) and Secretarybird(JSBirds/Secretarybird).

JSBird/demo is a demo scnene that uses all birds.
